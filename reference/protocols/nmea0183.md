# NMEA 0183 quick reference

## Basics
- Serial ASCII, typically 4800 8N1 (some talkers 38400 for high-speed AIS).
- Talker ID (2 chars), sentence formatter (3 chars), comma-separated fields, optional checksum after `*` (XOR of chars between `$` and `*`).

## Example sentences
- GPS position (RMC): `$GPRMC,123519,A,4807.038,N,01131.000,E,022.4,084.4,230394,003.1,W*6A`
- Depth (DBT): `$IIDBT,035.6,f,010.8,M,005.9,F*25`
- Heading (HDG): `$HCHDG,101.1,,,7.1,W*3C`
- AIS high-speed (38400 baud) VDM fragment: `!AIVDM,1,1,,A,15Muq@P000G?tO`B:`;HUwvN80<3,0*5C`

## Checklist
- Verify baud rate: 4800 vs 38400 (AIS).
- Check grounding and shield on long runs; avoid ground loops.
- Validate checksum when troubleshooting; XOR of bytes between `$`/`!` and `*`.
- Some talkers use `!` for encapsulated sentences (AIS).
